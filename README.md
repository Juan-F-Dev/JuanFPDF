# JuanFPDF

Sistema de diseño visual para la creación de documentos PHP mediante la librería FPDF.

## Descripcion
Este sistema facilita la creación de documentos PHP de manera gráfica, optimizando el flujo de trabajo y reduciendo el tiempo de desarrollo. La herramienta permite diseñar la estructura del PDF visualmente, eliminando la necesidad de gestionar manualmente coordenadas complejas en el código fuente.

## Tecnologias
- Frontend: React + Vite
- Estilos: Tailwind CSS
- Gestion de Estado: Zustand
- Salida: PHP (FPDF Library)

## Funcionalidades Principales
- Interfaz de arrastrar y soltar (Canvas) para la ubicación de componentes.
- Editor de propiedades de componentes para ajustes rápidos y sencillos.
- Exportador de código directo a formato PHP compatible con FPDF.

## Instalacion
El proyecto está diseñado para ser ejecutado mediante Docker.

1. Clonar el repositorio:
   git clone https://github.com/Juan-F-Dev/JuanFPDF.git


Nota: Se puede ejecutar localmente con "npm install" y "npm run dev" si se dispone del entorno de Node.js configurado.
